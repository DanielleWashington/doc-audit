# What is ConfigFlow?

ConfigFlow is a distributed configuration management tool designed for cloud-native applications.

It supports YAML, JSON, TOML, and HCL formats. It integrates with Kubernetes, Nomad, ECS, and 40+ platforms. Features include versioning, rollbacks, ACLs, audit logging, health checks, dynamic secrets, and service mesh support.

ConfigFlow has been adopted by over 3,000 organizations worldwide. It is open source under the MPL 2.0 license and maintained by a team of 12 full-time engineers.

Our roadmap includes Wasm plugin support, improved UI, and a new telemetry system planned for Q3.
